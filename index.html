BROSTEL - Find your perfect hostels.

BROSTEL is a comprehensive platform built using Node.js, Express.js, MongoDB, React.js, Redux Toolkit, and Tailwind CSS, designed to connect users with the perfect hostel or PG. Featuring separate user, vendor, and admin panels, BROSTEL includes a robust booking system, live location via Mapbox, subscription plans, and a real-time chat feature powered by Socket.IO. With its seamless functionality and verified listings, BROSTEL makes finding reliable accommodations convenient and stress-free, ensuring a smooth experience for both working professionals and students.

6XO BAGs - Ecommerce store.

6XO Bags is a dynamic e-commerce platform built entirely from scratch using Node.js, Express.js, MongoDB, and EJS. It features a comprehensive product management system, seamless cart and order management, and a robust admin dashboard with tools for tracking sales, generating detailed reports, and visualizing data through interactive sales charts. The store is equipped with secure Razorpay payment integration, ensuring a smooth and reliable checkout process. This project showcases my ability to design and develop fully functional e-commerce solutions, combining technical expertise with a user-centric approach.
